# configuration

To store my configuration files or scripts

### Usage

Install git first.

```sh
$ git clone https://github.com/rijn/configuration ~/configuration
$ sudo sh ~/configuration/deploy.sh
```

### Support OS

* OSX 10.9
* CentOS 7.2
* Ubuntu

### Dependencies

* zsh
* toilet
* vim
* libpng
* cmake
* curl
* wget
* Vumdle
* MacVim
